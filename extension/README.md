# Browser Extension

Unpacked Chrome/Edge extension that replaces colorful Figma web loading bars with a blue, neutral, or custom-color loader.

## Install

1. Open `chrome://extensions` or `edge://extensions`.
2. Enable developer mode.
3. Click "Load unpacked".
4. Select this `extension` folder.
5. Refresh `https://www.figma.com`.

Use the toolbar popup to choose the loader color.
